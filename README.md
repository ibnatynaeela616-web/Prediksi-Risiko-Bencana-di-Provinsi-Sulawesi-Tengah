# Prediksi Risiko Bencana di Provinsi Sulawesi Tengah
Proyek ini merupakan implementasi machine learning untuk tugas Statistika pada Program Studi Sistem Informasi, Universitas Tadulako. Model KNN digunakan untuk mengklasifikasikan tingkat risiko bencana alam menjadi tiga kategori yaitu Rendah, Sedang, dan Tinggi berdasarkan data historis kejadian bencana dari Badan Nasional Penanggulangan Bencana (BNPB) selama periode 2010 hingga 2024.

# Sumber Data
Badan Nasional Penanggulangan Bencana (BNPB) Provinsi Sulawesi Tengah
Periode 2010 – 2024
